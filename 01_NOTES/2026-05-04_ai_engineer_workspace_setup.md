# AI Engineer Workspace Setup (May 4)

## 1. Who is an AI Engineer?
An AI Engineer builds intelligent applications using Python, APIs, LLMs, databases, backend systems, and deployment tools. AI engineers do not only write ML code; they create complete working AI products.

## 2. Standard AI Project Workflow
Idea → Create Project Folder → Create Virtual Environment → Install Libraries → Write Code → Test → GitHub Push → Improve → Deploy

## 3. What is Virtual Environment?
A virtual environment is an isolated Python workspace for one project. It stores project-specific libraries separately so package versions do not clash with other projects.

## 4. Why GitHub Matters
GitHub stores code publicly/professionally, tracks progress, and becomes proof of portfolio for jobs.

- python-dotenv
